# CD Medium

The compact disc attached to this thesis contains the complete source code of the face detection and recognition system, together with the quantized (TensorFlow Lite) model weights, the configuration files, Makefile targets, and scripts used to run training and evaluation. Its directory layout follows the repository structure described in the System Manual (Appendix C).

The custom face datasets and the full-precision training checkpoints are not included on the medium: the datasets contain personal data (face images), and the checkpoints can be regenerated from the source code by running the training pipeline. Both are available from the author on request.
