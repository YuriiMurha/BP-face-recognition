# CD Medium

The compact disc attached to this thesis contains the complete source code of the face detection and recognition system, including the configuration files, Makefile targets, and scripts used to run training and evaluation. Its directory layout follows the repository structure described in the System Manual (Appendix C).

The custom face datasets and the trained model weights are not included on the medium: the datasets contain personal data (face images), and the models can be regenerated from the source code by running the training pipeline. Both are available from the author on request.
